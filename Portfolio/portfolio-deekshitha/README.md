# Deekshitha Manchikatla — Portfolio Website

A modern, responsive personal portfolio website built with pure HTML, CSS, and JavaScript.
No frameworks, no build tools, no npm required.

---

## 📁 Folder Structure

```
portfolio-deekshitha/
│
├── index.html          ← Main HTML page (open this in browser)
├── assets/
│   ├── style.css       ← All styles
│   ├── script.js       ← All interactivity
│   └── Deekshitha_Resume.pdf  ← (Add your resume PDF here)
└── README.md
```

---

## 🚀 How to Run in VS Code

### Step 1 — Open the folder
```
File → Open Folder → select "portfolio-deekshitha"
```

### Step 2 — Install Live Server (one-time, free)
1. Press **Ctrl + Shift + X** to open Extensions
2. Search for **"Live Server"** by Ritwick Dey
3. Click **Install**

### Step 3 — Launch the site
- Right-click `index.html` in the Explorer panel
- Select **"Open with Live Server"**
- Your browser opens at `http://127.0.0.1:5500`
- Every save auto-refreshes the browser ✅

## ✏️ Things to Customise

Open the files below and update these placeholders:

### index.html
| Placeholder | Replace with |
|---|---|
| `https://github.com/YOUR_USERNAME` | Your actual GitHub URL |
| `https://linkedin.com/in/YOUR_USERNAME` | Your actual LinkedIn URL |
| Project `href="#"` links | Your GitHub repo / live demo URLs |

### assets/ folder
- Drop your resume PDF inside `assets/` and name it `Deekshitha_Resume.pdf`
  - The **Download Resume** button in the nav will work automatically.

## 🌐 Deploy for Free (Optional)

### Netlify (easiest)
1. Go to https://netlify.com → Sign up free
2. Drag and drop the `portfolio-deekshitha` folder onto the Netlify dashboard
3. Done — you get a live URL like `https://deekshitha.netlify.app`

### GitHub Pages
1. Push the folder to a GitHub repository
2. Go to Settings → Pages → Source → main branch
3. Your site goes live at `https://yourusername.github.io/portfolio-deekshitha`

## 🎨 Colour Scheme

| Variable | Value | Usage |
|---|---|---|
| `--ink` | `#0d0d14` | Text, dark backgrounds |
| `--paper` | `#f5f2eb` | Page background |
| `--accent` | `#e8533a` | Coral red highlights |
| `--accent2` | `#3a7de8` | Blue highlights |
| `--muted` | `#6b6860` | Secondary text |

To change the colour theme, edit these variables at the top of `assets/style.css`.
